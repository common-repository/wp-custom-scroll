== Description ==

This plugin use for wordpress custom scroll on whole page.

= Features =
* Easy Installation
* It is user friendly and very easy to install.
* Zero coding skills required to set up this plugin.
* No need to settings in admin area

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh "Wp Custom Scroll"
3. install and activate the WordPress Plugin.
